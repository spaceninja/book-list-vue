# Book List Vue

Bootstrapped with Vite & Vue 3.

## Project setup

```
yarn
```

### Compiles and hot-reloads for development

```
yarn dev
```

### Compiles and minifies for production

```
yarn build
```

### Lints and fixes files

```
yarn lint
```

### Run tests

```
yarn test
```

## Dependency Note

Do not upgrade Jest to v27 until vue-jest updates to support it!
